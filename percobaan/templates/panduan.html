Panduan Penggunaan Aplikasi Sentilytics

📥 1. Download Ulasan dari Play Store

  • Masukkan App ID dari aplikasi (contoh: com.shopee.id)
  • Tekan tombol Download Ulasan CSV
  • File CSV akan otomatis terunduh

📊 2. Upload File CSV Tanpa Sentimen

  • Siapkan file CSV berisi kolom full_text (isi ulasan)
  • Upload file, lalu tekan Prediksi dan Unduh
  • File akan otomatis dianalisis dan hasilnya akan diunduh

🧠 3. Upload File dengan Sentimen (Analisis Naive Bayes)

  • Upload file CSV yang sudah memiliki kolom sentimen
  • Pilih opsi "Balancing sebelum split data" jika ingin distribusi seimbang
  • Tekan Analisis dan Tampilkan Hasil untuk melihat grafik dan evaluasi model

💬 4. Bantuan Chatbot Interaktif

  • Buka halaman Bantuan melalui sidebar (menu samping kiri)
  • Ketik pertanyaan seperti "Bagaimana cara upload CSV?" pada kolom chat
  • Tekan Kirim atau tombol Enter untuk mendapatkan jawaban
  • Chatbot akan memberikan bantuan langsung di layar

🌓 5. Tema Gelap / Terang

  • Gunakan tombol 🌙 atau 🌞 di pojok kanan atas untuk mengganti tema
